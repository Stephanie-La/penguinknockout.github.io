PROJECT DESCRIPTION

Discover the amazing world of physics in this fun, interactive penguin game!

Change the slingshot angle, direction, and force to send the penguins flying towards the targets!

Clear each level to become a master of elementary physics. For ages 10+

PROJECT BACKGROUND

Physics can be daunting - this game teaches middle schoolers the topic in a fun, easy-to-learn way.

Users will be able to pull the slingshot back at a distance and an angle, allowing the user to hit (or miss!) the target.

This game utilizes simulated physics that mimics the real world.
